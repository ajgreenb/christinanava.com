Step 1.
=======
Put all the PNGs you want into the media/ folder.
Give them number names (currently they are sorted
numerically, but we can easily change this if you
want.)

Step 2.
=======
Open your Terminal.app, and type

  cd ~/Desktop/christinanava.com
  python resize.py
  python generate.py
  cd _site
  python -m SimpleHTTPServer

Step 3.
=======
Your website is now running on your computer!
Go to http://localhost:8000/

Step 4.
=======
Change images, repeat.
